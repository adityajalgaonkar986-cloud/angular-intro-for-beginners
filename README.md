# What is Angular and Why is it Used for Frontend Development?

Angular is a **TypeScript‑based, open‑source front‑end framework** maintained by Google, designed to build **dynamic, scalable single‑page applications (SPAs)** for web and mobile. [web:1][web:20]  

This repository is a beginner‑friendly guide to:
- What Angular is  
- Why it is used for frontend development  
- Core concepts: Components, Data Binding, and TypeScript  
- Ready‑to‑use HTML demo  

### 🔧 Why Use Angular?

- **Component‑based architecture** – reusable UI blocks. [web:1][web:20]  
- **Two‑way data binding** – automatic sync between model and view. [web:1]  
- **TypeScript** – better tooling, type safety, and maintainability. [web:20]  
- **Built‑in features** – routing, forms, HTTP client, dependency injection. [web:20]
